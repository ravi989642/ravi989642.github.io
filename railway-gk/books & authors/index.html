Q. 9802 📋 RPF SI (06/01/2019) Shift 2
इनमें से कौन एक प्रसिद्ध खगोलीय विज्ञान पुस्तक है?
A सुश्रुत संहिता
B सूर्य सिद्धांत ✅
C चरक संहिता
D पंचतंत्र
🎯 Correct Answer
(B) सूर्य सिद्धांत
📖 Deep Explanation

सूर्य सिद्धांत एक प्राचीन भारतीय खगोलीय ग्रंथ है, जिसमें ग्रहों की गति, सूर्यग्रहण, चंद्रग्रहण, और समय गणना का वर्णन है। यह वैदिक काल की खगोल विज्ञान की प्रमुख पुस्तक है।

  • सुश्रुत संहिता → आयुर्वेद (शल्य चिकित्सा)
  • चरक संहिता → आयुर्वेद (चिकित्सा शास्त्र)
  • पंचतंत्र → नीति कथाएँ
⚡ Quick Memory Bytes (Exam Must)
सूर्य सिद्धांत किस विज्ञान से संबंधित है? → खगोल विज्ञान
सूर्य सिद्धांत का रचयिता किसे माना जाता है? → आर्यभट्ट (प्रमुख रूप से)
सुश्रुत संहिता किस विधा की पुस्तक है? → आयुर्वेद (शल्य चिकित्सा)
चरक संहिता का संबंध किससे है? → आयुर्वेद (चिकित्सा शास्त्र)
पंचतंत्र किस प्रकार की रचना है? → नीति कथाएँ
🧠 Memory Trick
"सूरज = खगोल" → सूर्य (सूरज) दिखता है आकाश (खगोल) में, इसलिए सूर्य सिद्धांत = खगोल विज्ञान!

"चरक = चिकित्सक" → दोनों 'च' से शुरू → चरक = चिकित्सा/डॉक्टर
"सुश्रुत = शल्य" → दोनों 'श' से शुरू → सुश्रुत = शल्य (सर्जरी)

Q. 9803 📋 RPF SI 2018 (10/01/2019) Shift 3
'अनब्रेकेबल: एन ऑटोबायोग्राफी' पुस्तक किसने लिखी है?
A वायु सिंह
B मैरी कॉम ✅
C शिव थापा
D विकास कृष्ण यादव
🎯 Correct Answer
(B) मैरी कॉम
📖 Deep Explanation

'अनब्रेकेबल: एन ऑटोबायोग्राफी' भारतीय महिला मुक्केबाज मैरी कॉम की आत्मकथा है। इसे उन्होंने सह-लेखिका सुधा मेनन के साथ लिखा। यह पुस्तक 2018 में प्रकाशित हुई।

  • मैरी कॉम छह बार की विश्व चैंपियन और ओलंपिक पदक विजेता हैं
  • यह पुस्तक उनकी प्रेरणादायक यात्रा को दर्शाती है
⚡ Quick Memory Bytes (Exam Must)
मैरी कॉम की आत्मकथा का नाम → अनब्रेकेबल: एन ऑटोबायोग्राफी
मैरी कॉम के सह-लेखक → सुधा मेनन
प्रकाशन वर्ष → 2018
मैरी कॉम का खेल → मुक्केबाजी
मैरी कॉम के विश्व खिताब → 6
🧠 Memory Trick
"मैरी = मुक्केबाजी" → दोनों 'म' से शुरू → मैरी कॉम = मुक्केबाजी

"अनब्रेकेबल = अटूट" → मुक्केबाजी में चोट लगती है पर वो अटूट रहीं, इसलिए किताब का नाम 'अनब्रेकेबल'!
